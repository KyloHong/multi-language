# Figma多语言工作流 - PRD

## 1. Product Overview

Figma多语言工作流是一个自动化工具，用于从Figma设计稿中提取文案并生成多语言文档。解决了传统手动扒文案、翻译效率低的问题，为产品设计师和本地化团队提供高效的工作流程。

## 2. Core Features

### 2.2 Feature Module
1. **首页/工作流入口**: Figma链接输入、工作流引导
2. **文案预览页**: 展示提取的文案、编辑功能
3. **品牌平台选择页**: 品牌和平台选择、多语言配置
4. **多语言输出页**: 翻译结果预览、文档下载
5. **AI校对页**: 重复检测、质量评估、修改建议

### 2.3 Page Details

| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | Figma上传模块 | 支持Figma链接输入、文件上传、Token配置 |
| 首页 | 工作流步骤指示器 | 显示当前步骤、已完成步骤 |
| 文案预览页 | 文案列表 | 展示提取的文案、节点信息、编辑功能 |
| 文案预览页 | 批量操作 | 批量选择、删除、导出 |
| 品牌平台选择页 | 品牌选择 | 下拉选择品牌（如AU、EU、US等） |
| 品牌平台选择页 | 平台选择 | 选择对应平台（如IB portal、Admin等） |
| 多语言输出页 | 翻译预览 | 多语言对照展示、编辑翻译 |
| 多语言输出页 | 文档导出 | JSON、CSV、Excel格式导出 |
| AI校对页 | 重复检测 | 检测重复文案、显示场景信息 |
| AI校对页 | 去重操作 | 相同场景自动去重、手动确认 |
| AI校对页 | 质量评估 | AI评分、修改建议列表 |

## 3. Core Process

用户流程：
1. 输入Figma链接或上传文件
2. 系统提取文案并展示预览
3. 用户选择品牌和平台
4. 系统生成对应多语言翻译
5. AI校对检测重复和质量
6. 用户确认并下载文档

```mermaid
flowchart LR
    A[输入Figma链接] --> B[提取文案]
    B --> C[文案预览与编辑]
    C --> D[选择品牌和平台]
    D --> E[生成多语言翻译]
    E --> F[AI校对]
    F --> G[下载多语言文档]
```

## 4. User Interface Design

### 4.1 Design Style
- 主色调：深蓝色 (#1e40af) + 青色 (#06b6d4)
- 辅助色：中性灰色系
- 按钮风格：圆角矩形，悬停有轻微阴影
- 字体：无衬线现代字体
- 布局风格：卡片式布局，清晰的视觉层级
- 图标：Lucide图标库，简洁线条风格

### 4.2 Page Design Overview

| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | Figma上传模块 | 大型输入框、拖拽区域、帮助提示、渐变背景 |
| 文案预览页 | 文案列表 | 表格布局、搜索框、筛选器、分页、动画过渡 |
| 品牌平台选择页 | 选择器 | 卡片式选择、品牌Logo展示、悬停效果 |
| 多语言输出页 | 翻译预览 | 分栏布局、语言标签、编辑模式切换 |
| AI校对页 | 检测结果 | 进度条、警告提示、建议卡片、操作按钮 |

### 4.3 Responsiveness
桌面优先设计，适配平板和移动设备，关键操作保持可访问性。

### 4.4 Animation
- 页面加载时的渐入动画
- 卡片悬停时的轻微上浮效果
- 步骤切换时的滑入过渡
- 按钮点击时的缩放反馈
